Capital One releases VulnHunter, an open-source AI tool that finds software flaws before hackers do

VentureBeat – AI · Topic: AI/Cloud/Quantum

  • Who: Developed and released by Capital One, a leading financial institution, this AI tool leverages advanced models like Anthropic's Claude to secure software globally.

Brex built its AI agent policy by watching what agents actually do, not by writing rules first

VentureBeat – AI · Topic: AI/Cloud/Quantum

  • What it is: Brex developed CrabTrap, an open-source HTTP/HTTPS proxy and policy engine, which intelligently governs AI agent behavior by intercepting network traffic and utilizing an LLM-as-a-judge.
  • Who: Brex, a financial technology company, developed an innovative AI agent governance platform, championed by co-founder Pedro Franceschi, which has been featured by VentureBeat.
  • What it does: CrabTrap intercepts AI agent network requests, applying dynamic policy rules and an LLM-as-a-judge to approve or deny actions based on learned behavioral patterns, instead of static pre-set guidelines.
  • Why it matters: This solution fundamentally shifts AI agent governance from SDK-level permissions to a centralized, adaptive network control plane, enhancing enterprise confidence for broad, secure autonomous agent deployment.
  • Strategic view: TMT executives should recognize this new paradigm for AI agent governance, enabling secure, scalable enterprise adoption by shifting control to the network layer, fostering trust, and maximizing autonomous capabilities.
