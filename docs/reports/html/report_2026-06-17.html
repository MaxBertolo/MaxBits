Z.ai’s open-weights GLM-5.2 beats GPT-5.5 on multiple long-horizon coding benchmarks for 1/6th the cost

VentureBeat – AI · Topic: AI/Cloud/Quantum

  • What it is: Z.ai's GLM-5.2 is a 753-billion parameter open-weights large language model engineered specifically to dominate long-horizon autonomous coding and engineering tasks, available via MIT license.
  • Who: Chinese AI startup Z.ai, formerly Zhipu AI, developed and released this advanced large language model, making it available globally on Hugging Face and via API.
