Cerebras says its chips run a trillion-parameter AI model nearly 7 times faster than GPU clouds

VentureBeat – AI · Topic: AI/Cloud/Quantum

  • What it is: Cerebras Systems announced its wafer-scale chips run a trillion-parameter AI model, Kimi K2.6, significantly faster for inference than existing GPU cloud providers.
  • Who: Cerebras Systems demonstrated its chips' performance with Moonshot AI's Kimi K2.6 model, benchmarked by Artificial Analysis, challenging Nvidia's GPU cloud dominance.
  • What it does: Cerebras' wafer-scale architecture delivers nearly 1,000 tokens per second for trillion-parameter AI inference, providing a 6.7x to 29x speed improvement over GPU solutions.
  • Why it matters: This breakthrough offers enterprises a vastly more performant and potentially cost-effective alternative for deploying large AI models, addressing current capacity and latency challenges.
  • Strategic view: TMT executives must assess alternative AI hardware like Cerebras' wafer-scale chips for future large model deployments, considering performance gains and geopolitical implications of model origins.

Cohere cracks lossless quantization and native citations with first full Apache 2.0 licensed open model Command A+

VentureBeat – AI · Topic: AI/Cloud/Quantum

  • What it is: Cohere has released Command A+, a 218-billion-parameter Sparse Mixture-of-Experts language model, optimized for complex reasoning and agentic workflows under a permissive Apache 2.0 license.
  • Who: Cohere, a leading Canadian AI lab recently merged with Aleph Alpha and co-founded by Aidan Gomez, developed and released Command A+.
  • What it does: Command A+ features lossless 4-bit quantization for extreme hardware efficiency, native citation generation for enterprise trust, and enhanced multilingual tokenizer support for global deployments.
  • Why it matters: This model allows enterprises to deploy powerful, secure, and verifiable AI capabilities on-premise with significantly reduced compute resources, lowering operational costs and increasing data control.
  • Strategic view: Command A+ champions sovereign AI by enabling C-levels to achieve high-performance, cost-efficient, and auditable AI within their secure environments, accelerating trusted enterprise-wide adoption and innovation.

GitHub confirms 3,800 internal repos stolen through poisoned VS Code extension as supply chain worm hits Microsoft’s Python SDK

VentureBeat – AI · Topic: AI/Cloud/Quantum

  • What it is: GitHub confirmed that approximately 3,800 internal repositories were stolen after an employee installed a poisoned VS Code extension, leading to an infrastructure intelligence leak.
  • Who: GitHub, a Microsoft-owned platform, was breached by the financially motivated threat group TeamPCP (UNC6780), specializing in complex software supply chain attacks.
  • What it does: The compromised VS Code extension allowed attackers to exfiltrate critical GitHub-internal repositories, containing sensitive infrastructure configurations, deployment scripts, and API schemas.
  • Why it matters: This breach is an infrastructure intelligence leak, not just a data breach, enabling attackers to leverage stolen secrets and configurations for future, more sophisticated supply chain attacks.
  • Strategic view: C-level executives must prioritize robust software supply chain security, emphasizing employee device hygiene, rigorous third-party extension vetting, and advanced provenance verification against sophisticated, multi-vector attacks.
