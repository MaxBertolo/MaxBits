Meituan open sources LongCat-2.0, the 1.6T, near-frontier agentic coding model that's been leading OpenRouter — trained entirely on Chinese chips

VentureBeat – AI · Topic: AI/Cloud/Quantum

  • What it is: LongCat-2.0 is a 1.6-trillion-parameter, near-frontier agentic coding model open-sourced by Meituan, previously known as the top-performing "Owl Alpha" on OpenRouter charts.
  • Who: Chinese delivery app company Meituan developed and open-sourced LongCat-2.0, a powerful AI model that has been leading global developer charts.
  • What it does: This agentic coding model provides autonomous software engineering capabilities with a 1-million-token context window, aiming to disrupt closed-source enterprise AI at competitive, often free, pricing.
  • Why it matters: LongCat-2.0's significance lies in its top-tier performance on developer charts and its successful training entirely on over 50,000 Chinese ASICs, demonstrating AI independence from Nvidia GPUs.
  • Strategic view: This open-source release signals a critical shift towards hardware independence in frontier AI development, allowing Chinese tech to competitively challenge global GPU monopolies and proprietary models in the enterprise software market.

DeepSeek open sources DSpark, a new framework to speed up LLM inference by up to 85%

VentureBeat – AI · Topic: AI/Cloud/Quantum

  • What it is: DeepSeek open-sourced DSpark, a new MIT-licensed framework designed to significantly speed up Large Language Model (LLM) inference without changing the underlying model's output.
  • Who: DeepSeek, a Chinese open-source AI firm, developed and released DSpark, making it broadly available to developers, researchers, and commercial enterprise operations.
  • What it does: DSpark utilizes speculative decoding, where a smaller 'scout' model anticipates several next tokens for the main LLM to quickly verify, generating text up to 85% faster.
  • Why it matters: This innovation addresses a critical AI deployment challenge by making LLMs more hardware-efficient and responsive, improving user experience while lowering operational costs for enterprises.
  • Strategic view: DSpark democratizes access to faster, more cost-effective LLM deployment by reducing inference overhead, accelerating AI product cycles, and intensifying competition among TMT providers leveraging open models.

The attack that hijacked Claude Code came through Sentry. Datadog, PagerDuty, and Jira have the same exposure.

VentureBeat – AI · Topic: AI/Cloud/Quantum

  • What it is: An attack vector called "agentjacking" exploits AI coding agents by injecting malicious instructions through trusted diagnostic data sources, causing them to execute unauthorized code with developer privileges.
  • Who: Tenet Security disclosed the vulnerability affecting AI coding agents like Claude Code, Cursor, and Codex, with CrowdStrike and other security vendors now addressing runtime protection.
  • What it does: The attack leverages public credentials for services like Sentry to inject fake error reports, which AI agents then execute as trusted commands, bypassing traditional security defenses completely.
  • Why it matters: This represents a critical, systemic security flaw that current enterprise defenses fail to detect, enabling attackers to gain full developer privileges and exfiltrate sensitive data without triggering alerts.
  • Strategic view: C-level executives must prioritize continuous runtime security and granular action-level authorization for AI agents, treating them with stricter controls than humans to mitigate this profound new systemic risk.
